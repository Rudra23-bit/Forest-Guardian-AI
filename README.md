# 🌲 Forest Guardian AI

Forest Guardian AI is an AI-powered forest incident reporting platform designed to support citizens, forest officers, NGOs, and volunteers in reporting environmental incidents. The application uses AI-driven risk assessment and workflow automation to prioritize incidents, generate real-time alerts, and maintain a centralized incident database for efficient monitoring and response.

## 🚀 Features

* AI-powered incident risk assessment
* Real-time forest incident reporting
* GPS location capture
* Image upload support
* Automated workflow integration with n8n
* Email notifications for high-risk incidents
* Google Sheets integration for centralized data management
* Responsive and user-friendly interface
* Secure API-based communication

## 🛠️ Technology Stack

**Frontend**

* React
* TypeScript
* Tailwind CSS
* TanStack Router

**Automation & Integration**

* n8n
* Google Sheets API
* Gmail API
* AI Agent (LLM)

## 📊 Workflow

```text
User
   │
   ▼
Forest Guardian AI Web Application
   │
   ▼
n8n Webhook
   │
   ▼
AI Risk Assessment
   │
   ├── High-Risk Incident → Email Alert
   └── Standard Incident → Google Sheets Database
```
